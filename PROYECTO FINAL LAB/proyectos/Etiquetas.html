Lista de Etiquetas HTML, Descripción y Ejemplo
Etiqueta Descripción Ejemplo
<html> Define el documento como HTML. Es la raíz del documento. <html lang="es"> <head> <title>Ejemplo</title> </head> <body> <h1>Hola</h1> </body> </html>
<head> Contiene metadatos, como enlaces a hojas de estilo o scripts, y el título de la página. <head> <title>Título</title> <meta charset="UTF-8"> </head>
<title> Define el título que aparece en la pestaña del navegador. <title>Mi Página Web</title>
<style> Contiene reglas CSS para el estilo de la página. <style> body { background-color: lightblue; } </style>
<link> Vincula archivos externos como hojas de estilo o fuentes. <link rel="stylesheet" href="styles.css">
<meta> Proporciona metadatos como el conjunto de caracteres o la descripción de la página. <meta charset="UTF-8"> <meta name="description" content="Descripción de la página">
<script> Incrusta o enlaza scripts (JavaScript). <script src="app.js"></script>
<noscript> Proporciona un mensaje o contenido alternativo si JavaScript está deshabilitado. <noscript>JavaScript está desactivado.</noscript>
<body> Contiene el contenido visible de la página. <body> <h1>Contenido visible</h1> </body>
<header> Define un encabezado para un documento o sección. <header> <h1>Mi Sitio Web</h1> </header>
<nav> Define una sección de navegación. <nav> <a href="#inicio">Inicio</a> <a href="#contacto">Contacto</a> </nav>
<main> Contiene el contenido principal del documento. <main> <h2>Sección principal</h2> </main>
<article> Define un contenido independiente y auto-contenido. <article> <h2>Artículo</h2> <p>Este es un artículo.</p> </article>
<section> Define una sección de contenido relacionada. <section> <h2>Sección 1</h2> <p>Contenido de la sección 1.</p> </section>
<aside> Define contenido secundario o complementario, como una barra lateral. <aside> <h2>Información adicional</h2> <p>Detalles adicionales.</p> </aside>
<footer> Define el pie de página. <footer> <p>Copyright © 2024</p> </footer>
<h1> - <h6> Define encabezados, donde <h1> es el más importante y <h6> el menos importante. <h1>Título Principal</h1> <h2>Subtítulo</h2>
<p> Define un párrafo. <p>Este es un párrafo.</p>
<br> Inserta un salto de línea. Este es un texto<br>y este es otro en una nueva línea.
<hr> Inserta una línea horizontal. <hr>
<pre> Define texto preformateado, conservando espacios y saltos de línea. <pre>Texto preformateado. Con espacios.</pre>
<blockquote> Define una cita en bloque. <blockquote>Cita textual larga.</blockquote>
<ol> Define una lista ordenada. <ol> <li>Elemento 1</li> <li>Elemento 2</li> </ol>
<ul> Define una lista desordenada. <ul> <li>Elemento 1</li> <li>Elemento 2</li> </ul>
<li> Define un elemento de lista. <li>Elemento de lista</li>
<dl> Define una lista de definiciones. <dl> <dt>Término</dt> <dd>Definición del término.</dd> </dl>
<a> Define un hipervínculo. <a href="https://www.ejemplo.com">Visitar Ejemplo</a>
<img> Inserta una imagen. <img src="imagen.jpg" alt="Descripción de la imagen">
<table> Define una tabla. <table> <tr> <td>Celda</td> </tr> </table>
<form> Define un formulario HTML. <form> <input type="text"> <input type="submit"> </form>
